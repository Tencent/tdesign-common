# 进度条 Progress

## 组件设计指南

### 何时使用

当有一项系统任务正在进行，需要向用户展示该任务的当前进度时使用。

### 与页面布局相关

##### 当展示页面、模块的加载进度时，通常进度条放置在页面、模块的正中间。
