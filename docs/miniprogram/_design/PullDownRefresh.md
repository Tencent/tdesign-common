# 下拉刷新 PullDownRefresh

## 组件设计指南

### 何时使用

当需要对页面信息进行整体刷新或加载更多同类信息时使用。

### 组件搭配使用

##### 与输入类组合使用，用于帮助用户完成一些快捷的输入或选择；但不建议一个对话框内有多种不同的输入类组件。
