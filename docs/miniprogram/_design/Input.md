# 输入框 Input

## 组件设计指南

### 何时使用

需要进行文字输入，且需填写的字数较少时使用。

### 组件搭配使用

##### 呈现在单元格中，可与按钮组件相结合，点击按钮触发相关功能。

### 常见用法

##### 通常使用在表单中进行文字填写，填写状态下可在右侧显示清空输入的按钮，供用户快速清空已输入的内容。
