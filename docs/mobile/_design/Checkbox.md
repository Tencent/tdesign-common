# 多选框 Checkbox

## 组件设计指南

### 何时使用

当用户需要从一组数据选择多个选项时使用。

### 组件搭配使用

##### 多选框与[索引](./indexes)、[搜索框](./search)组合使用，便于用户从大量的选项中，快速地找到目标选项。
