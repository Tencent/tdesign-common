### 头像类型

头像提供了 3 种不同类型的头像：图标头像、图片头像、字符头像

{{ type }}

### 头像形状

头像默认支持两种形状：`circle`、`round`，用户也可自定义设置头像形状

{{ shape }}

### 头像大小

头像默认支持三种大小：`small`、`medium`、`large`，用户可自定义设置大小

{{ size }}

### 字符头像大小自适应

头像支持字符自适应，即字符长度过长时，头像可自动调整字符以便呈现完整内容

{{ adjust }}
